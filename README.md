#  Wine Quality Prediction using Machine Learning

##  Overview

This project focuses on predicting the quality of red wine based on its physicochemical properties using Machine Learning techniques. The model analyzes various chemical attributes such as acidity, sugar content, pH level, and alcohol percentage to estimate wine quality.

---

##  Features

* Predicts wine quality based on input parameters
* Uses Machine Learning algorithms for regression analysis
* Supports real-time prediction with user-provided data
* Data preprocessing and feature selection included
* Model evaluation using performance metrics

---

##  Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

##  Machine Learning Models

* Linear Regression
* Random Forest Regressor (optional enhancement)

---

##  Dataset

The dataset used in this project contains physicochemical properties of red wine, including:

* Fixed Acidity
* Volatile Acidity
* Citric Acid
* Residual Sugar
* Chlorides
* Free Sulfur Dioxide
* Total Sulfur Dioxide
* Density
* pH
* Sulphates
* Alcohol

File Name: Winequality Red.csv

---

##  How It Works

1. Load the dataset
2. Preprocess and clean the data
3. Split into training and testing sets
4. Train the machine learning model
5. Evaluate model performance
6. Predict wine quality based on new input

---

##  Model Performance

* Evaluated using metrics like:

  * Mean Squared Error (MSE)
  * R² Score
